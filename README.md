# SyntaxMasters

Live Website: https://syntaxmasters.vercel.app/

## Overview
SyntaxMasters is a modern web platform designed for developers to collaborate, learn programming concepts, and work on development projects together.

The platform provides a clean and interactive interface where users can explore coding resources, improve their development skills, and connect with other programmers.

---

## Features
- Modern and responsive UI
- Developer collaboration platform
- Coding resources and learning materials
- Fast cloud deployment using Vercel
- Accessible on desktop and mobile devices

---

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS

Backend / Services:
- Supabase

Deployment:
- Vercel

Version Control:
- Git & GitHub

---

## Installation

Clone the repository

```bash
git clone https://github.com/Pravinsaichiday/syntax-masters.git
```

Go to project folder

```bash
cd syntax-masters
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

---

## Deployment

The project is deployed using **Vercel**.

Steps:
1. Push code to GitHub
2. Import repository in Vercel
3. Click Deploy
